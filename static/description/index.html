Sale Order Approval
This module will add two level approval on sale order, only assigned user would be apple to approve sale order.

Features

  • User will submit quotation for approval
  • Maintain "waiting for approval" status with every quotation
  • Assign security group to manager
  • Manager can approve sale order

Submit Quotation/Order for approval

Assign approver security group to manager

Maintain "Waiting for approval" status

Manage can approve the sale order
